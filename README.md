# Vehicle Location Log v3

Mobile-friendly static prototype for logging and searching the latest known vehicle location within a dealership, compound, or storage area.

## v3 changes

- Added Mileage field to Check In
- Added Parking Location dropdown to Check In
- Added Parking Location dropdown to Update Location
- Added parking options: Sales, Budget, Parking 1–7, Other
- Displayed mileage in vehicle details and movement history
- Displayed parking location in vehicle details, movement history, and recent activity
- Displayed IN / OUT status in Recent Updates
- Limited Recent Updates to latest 10 records
- Updated main heading to "Track Vehicle Movements and Locations"
- Updated description to better reflect real dealership workflow
- Replaced demo/localStorage badge wording with "Vehicle Tracking Prototype"
- Removed GPS explanatory text from the form
- Renamed Recent Activity section to Vehicle Activity / Recent Updates
- Fixed Update Location panel so it hides when switching tabs
- Kept one-click Mark Out behaviour
- Kept Mark Out hidden when vehicle is already OUT

## Important

This is still a static prototype.

- No backend
- No shared database
- No customer information
- No dealership system integration
- Records are stored only in the current browser using localStorage
