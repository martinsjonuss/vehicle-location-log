# Vehicle Location Log

Mobile-friendly web application for logging and searching vehicle locations within a dealership, compound, or storage area.

## Current MVP Features

- Log a vehicle registration
- Select current vehicle location
- Add optional notes
- Search vehicles by registration
- Store data locally in the browser using localStorage

## Planned Features

- Edit vehicle location
- Vehicle movement history
- Mileage recording
- Damage/walk-around notes
- User login
- Shared database
- Admin dashboard

## Tech Stack

Current version:

- HTML
- CSS
- JavaScript

Possible future version:

- React + TypeScript
- REST API
- PostgreSQL
- Authentication
